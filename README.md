# Desafio--AWS-Step-Functions

O AWS Step Functions é uma funcionalidade da AWS no formato de fluxo, na qual os usuários são capazes de automatizar diversos processos, unindo validações e as demais ferramentas contidas no console.

- Vantagens
    Fácil visualização do processo
    Utilização de diversas ferramentas AWS em um só lugar
    Ramificação e Execução Paralela


- Integrações de serviços da AWS
O Step Functions se conecta a mais de 200 serviços da AWS. As mais comuns incluem lambda, dynamoDB, SQS, SNS
